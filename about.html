Syfte

Denna webbplats används för att visa, lägga till och radera arbetserfarenheter. Webbplatsen konsumerar en REST-baserad webbtjänst med hjälp av Fetch API.

Databas

Webbtjänsten använder MongoDB som NoSQL-databas. Backend är byggd med Node.js, Express och Mongoose.

Slutsats

Genom uppgiften har jag lärt mig hur frontend och backend kan hållas separerade. Jag har också lärt mig hur man använder Fetch API för att hämta, skicka och radera data från en REST-webbtjänst.
